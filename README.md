# Credit Risk Analysis & Default Prediction

## 📌 Problem Statement
Financial institutions face losses due to customer defaults.  
This project identifies high-risk customers using data analysis and machine learning.

## 🔍 Key Insights
- Repayment delay is the strongest predictor of default
- Lower credit limit customers show higher risk
- Demographic features have minimal impact

## 🤖 Model
- Logistic Regression
- ROC-AUC: 0.74
- Recall improved from 22% → 56%

## 📊 Dashboard
Power BI dashboard built for interactive analysis

## 🛠 Tools Used
- Python (Pandas, Scikit-learn)
- SQL
- Power BI

## 💼 Business Impact
- Reduced missed defaulters by ~45%
- Improved credit risk detection
